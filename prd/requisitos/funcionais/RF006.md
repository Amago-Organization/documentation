# RF006

Fazer Logout

## Descrição: 

O sistema deve permitir que os usuários já registrados possam fazer logout, ou seja, sair do sistema.

## Atores: 

Usuário Autenticado  

## Requisito de Dependência: 

[RF002](./RF002.md)

## Dados de Entrada:

- Sem dados de entrada

## Dados de Saída

- Todos os dados de usuário, com exceção de password;
