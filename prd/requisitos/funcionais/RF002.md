# RF002

Fazer Login

## Descrição: 

O sistema deve permitir que um usuário já cadastrado faça login.

## Atores: 

Usuário Não Autenticado  

## Requisito de Dependência: 

[RF001](./RF001.md)

## Dados de Entrada:

| Nome      | Tipo de Dado    | Obrigatório | Propriedade/Regra               |
|-----------|-----------------|-------------|---------------------------------|
| email     | String          | Sim        | Formato de e-mail, Máximo 225 caracteres                |
| password  | String          | Sim        | Entre 6 e 10 caracteres, deve conter ao menos 1 caracter maúsculo, 1 caracter especial, 1 caracter minúsculo, 1 número          |

## Dados de Saída

- Token de autenticação.
